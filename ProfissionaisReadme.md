# Readme Profissional

nesse projeto profissionais vão ter a liberdade de se cadastrar para poder prestar serviços, colocando seus horarios e seus serviços disponiveis para os clientes puderem encontralos.

2. celular
3. email
4. cpf
5. dataNascimento
6. cidade
7. estado
8. pais
9. rua
10. numero
11. bairro
12. cep
13. complemento
14. senha
15. Salario

## Funções possiveis

1. Cadastrar
2. Editar
3. Excluir
4. Recuperar senha
5. Pesquisar
